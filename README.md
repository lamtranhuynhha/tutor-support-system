# Tutor Support System

A comprehensive platform connecting students with tutors for personalized learning experiences.

## Features

- **Tutor Management**: Browse and search for qualified tutors
- **AI-Powered Matching**: Smart tutor-student matching using vector embeddings
- **Session Scheduling**: Easy booking and management of tutoring sessions
- **Interactive Learning**: Real-time communication tools for effective learning

## Tech Stack

- **Backend**: Node.js, Express, MongoDB
- **Frontend**: React, Vite, Tailwind CSS
- **AI/ML**: Vector embeddings for semantic search
- **Authentication**: JWT, OAuth2

## Getting Started

1. Clone the repository
2. Install dependencies: `pnpm install`
3. Set up environment variables
